Python Pandas之读取MySQL数据库

发表于 2016-12-11   |   分类于 学代码的Tracy

Mark一下用Python读取MySQL数据库的过程，以备使用。

输入：

import pandas as pd
import MySQLdb
# 输入数据库信息
mysql_cn= MySQLdb.connect(host='127.0.0.1', port=3306, user='root', passwd='wobushimima', db='crashcourse')
# 读取数据库中的products表，赋值给df
df = pd.read_sql('select * from products;', con=mysql_cn)    
mysql_cn.close()

# 打印df表
print '======================df======================'
print df
# 使用describe()对df表进行描述统计
print '=================describe it=================='
print df.describe()

附：mac下的环境配置

首先，当然是要安装pandas包啦

$ pip install pandas

这个过程比较久，可以去吃点小零食~

另外还要安装MySQLdb，命令如下：

$ pip install MySQL-python

然而在mac下会出现错误提示：

EnvironmentError: mysql_config not found

最简单的解决方式是，在pip install前加上这一行：

$ sudo ln -s /usr/local/mysql/bin/* /usr/bin

这样，环境就配置好了。

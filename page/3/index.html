人的一生在追求什么

发表于 2016-03-19   |   分类于 湿货

我在比现在还年少的岁月里，也思考过这样一个问题：人生的意义是什么？

当时我还不知道马斯诺需求层次理论，只是大致地想象了一下，人们一生普遍追求的，大概是事业、财富、爱情这些东西。哲学家比较厉害，他们人生的意义在于，思考人生有什么意义……

后来流行起了“自私的基因”的概念，按照生物进化的视角，人生的终极意义是传承。

不过，基因的传承理论并不能解释诸多事情，比如丁克，比如领养，比如对少数人来说，传承基因听起来远没有传播思想来的诱人。

再后来我琢磨出了一个似乎可以囊括所有种类的追求的通式：多巴胺。

发表于 2016-03-12   |   分类于 读书

几个月前看了《异类》这本书，作者是Malcolm Gladwell。作者对普世的成功人士进行了研究，分析总结出了一些观点。书中的数据更详实，有兴趣的朋友请阅读原作。

这本书对我看待问题的方式产生了极大影响。今天特将我印象比较深刻的观点列出来，并引用书中例子，整理成此份读书笔记。

马太效应

20世纪80年代，有一位名叫罗杰·巴恩斯利的加拿大心理学家注意到：加拿大顶级曲棍球运动员中，1月份出生的球员在人数上占有压倒性优势，远远超过其他月份出生的球员。排在第二位的呢？2月份。第三位的？3月份。经过更多的统计分析，巴恩斯利发现，这是因为：在加拿大，曲棍球联赛法定的注册时间是1月1日。
